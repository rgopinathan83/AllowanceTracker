## 📱 Overview

Allowance Tracker is a clean, minimalist Android app designed to help parents track their children's allowances, spending, and savings goals. Built with modern Android technologies, it provides a simple yet powerful way to teach kids financial literacy.

### 🎯 Why This App?

- **No Bloat**: Just the essential features you need
- **Privacy First**: All data stays on your device
- **Kids-Friendly**: Simple interface that children can understand
- **Real-World Skills**: Teaches budgeting, saving, and financial responsibility

---

## ✨ Features

### Core Features
- ✅ **Child Profiles** - Add and manage multiple children
- ✅ **Allowance Tracking** - Add income and expense transactions
- ✅ **Balance Management** - Real-time balance updates
- ✅ **Transaction History** - Complete audit trail with search
- ✅ **Edit/Delete Transactions** - Full control over records
- ✅ **Recurring Allowance** - Automated weekly allowances

### Savings Goals 🎯
- ✅ **Create Goals** - Set savings targets with custom icons
- ✅ **Track Progress** - Visual progress bars with percentages
- ✅ **Allocate Funds** - Move money from balance to goals
- ✅ **Edit Goals** - Update target amounts and descriptions
- ✅ **Delete Goals** - Remove completed or unwanted goals
- ✅ **Goal Completion** - Automatic celebration when goals are met

### User Experience
- ✅ **Material Design 3** - Modern, clean interface
- ✅ **Dark/Light Theme** - Automatic theme switching
- ✅ **Search Transactions** - Find transactions quickly
- ✅ **Statistics Dashboard** - Overview of goals and progress
- ✅ **Responsive Layout** - Works on phones and tablets

## 🚀 Installation

### 📥 Download APK

1. Go to [Releases](https://github.com/yourusername/allowance-tracker/releases)
2. Download the latest APK
3. Enable "Install from Unknown Sources" in your device settings
4. Install the APK

### 🛠️ Build from Source

#### Prerequisites
- Android Studio Hedgehog or later
- JDK 17 or higher
- Android SDK 34

#### Steps
```bash
# 1. Clone the repository
git clone https://github.com/yourusername/allowance-tracker.git

# 2. Open in Android Studio
cd allowance-tracker
open -a "Android Studio" .

# 3. Sync Gradle
# Click "Sync Now" when prompted

# 4. Build and Run
# Click the Run button or press Shift+F10
